## Installation

Make sure you have **Node.js** installed. You can check your version with:

```sh
node -v
npm -v
```

### **1. Clone the Repository**
```sh
git clone https://github.com/abrahamgregorius/haventwined_frontend
cd haventwined_frontend
```
### **2. Install Dependencies**
```sh
npm install
```
### **3. Run the app**
```sh
npm run dev
```
